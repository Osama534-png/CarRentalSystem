## ⚙️ Prerequisites & Installation

### Prerequisites
- Visual Studio 2022 (or suitable IDE)
- .NET SDK (Compatible version)
- SQL Server

### Installation & Run

1. Clone the repository:
```bash
git clone https://github.com/Osama534-png/WebApplication6.git 
```
Open the project in Visual Studio.
Update the DefaultConnection string in the appsettings.json file according to your SQL Server configuration.
Open the Package Manager Console (PMC) and run the following command to create the database:Update-Database
